A simple vue3 js application to display a list of items and bonuses in a game.

Contains main app.js file with logic.
Styles are place in styles.css.
HTML code is in index.html.

Main bonuses.json file contains main data configuration and references to individual bonuses files.

Code principles to follow:
minimum amount of code
reuse existing code instead of wring new one
follow best practices

Do not use whole-file rewrite commands. Use apply_patch only. Preserve UTF-8 encoding exactly.
